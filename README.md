# Agent Design Aesthetics

**Choose the visual language. Preserve the intent. Make the interface feel built on purpose.**

Agent Design Aesthetics is a schema-shaped AI agent repo for turning UI inspiration, design patterns, visual tips, component libraries, and aesthetic systems into reusable instructions for AI coding agents.

This repo is shaped after `schema-agent`: purpose-first, rule-aware, feature-ticket driven, repo-native, journaled, modular, and reusable.

## Purpose

Agent Design Aesthetics helps AI agents create better interfaces by converting design knowledge into practical build instructions.

It answers:

```text
What is the product trying to do?
What interface surface is being built?
What visual language fits the intent?
Which component systems should support it?
What prompt should the coding agent receive?
How do we validate that the result is polished, readable, accessible, and useful?
```

## Core Idea

A design agent should not only say "make it look modern."

A design agent should have:

```text
Intent
Purpose
Rules
Firing Order
Knowledge Sources
Feature Tickets
Prompt Recipes
Design Pattern Catalog
Validation
Release Memory
Repo Structure
```
