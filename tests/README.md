# Tests

## Purpose

This folder stores validation notes and test prompts for Agent Design Aesthetics.

## First Test

Ask the agent:

```text
Create a design-pattern table for a modern AI dashboard and explain how a coding agent should apply each pattern.
```

Expected output:

- table of patterns,
- explanation of what each pattern is,
- links or source names,
- AI-agent application instructions,
- accessibility requirements,
- validation checklist.

## Rule

Every major output mode should eventually have a test prompt and expected output shape.
