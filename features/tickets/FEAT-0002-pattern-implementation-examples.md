## Purpose

Add implementation examples that show how Design Aesthetics Agent recommendations turn into buildable UI instructions across common stacks and surfaces.

## Scope


## Acceptance Criteria

- Each example names the surface.

- Each example includes component sources.
- Each example includes an agent-ready prompt.
- Each example includes accessibility checks.
- Each example includes validation notes.
