# Visual Media Analysis Checklist

## Purpose

This checklist helps the agent convert a screenshot, video, reel, mockup, or visual reference into structured Design Aesthetics knowledge.

## Analysis Steps

```text
1. Identify the visible interface surface.
2. List every noticeable visual pattern.
3. Separate layout patterns from style effects.
4. Identify typography choices.
5. Identify spacing and density.
6. Identify color, contrast, and depth.
7. Identify motion or interaction cues.
8. Identify component families.
9. Map each item to source links or source names.
10. Convert each item into AI-agent application instructions.
```

## Important Distinction

Do not confuse:

| Layout | Style | Component | Motion |
| --- | --- | --- | --- |
| Bento Grid | Glassmorphism | Card | Fade-in |
| Split Hero | Claymorphism | Button | Hover lift |
| Sidebar Shell | Neomorphism | Modal | Page transition |

## Validation

- [ ] Did the agent name the surface?
- [ ] Did the agent explain each visual item?
- [ ] Did the agent provide source links where possible?
- [ ] Did the agent convert each item into a prompt-ready instruction?
- [ ] Did the agent note anything uncertain?

## Rule

When visual media is unclear, describe the uncertainty instead of guessing with confidence.
