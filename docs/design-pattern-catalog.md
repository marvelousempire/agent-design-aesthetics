# Design Pattern Catalog

## Purpose

This catalog explains major interface aesthetics and how AI agents should apply them.

| Item | What It Is | Why It Matters | Where To Find It | How AI Agents Apply It |
| --- | --- | --- | --- | --- |
| Bento Grid | A modular card layout with mixed card sizes and strong hierarchy. | Excellent for dashboards, AI tools, feature pages, and overview screens. | https://bentogrids.com | Ask the coding agent to create a responsive card grid with varied card sizes, clear hierarchy, and grouped features. |
| Glassmorphism | Frosted-glass panels using transparency, blur, borders, and soft depth. | Creates premium, futuristic, AI/SaaS-style interfaces. | Search official design references and modern UI libraries. | Use translucent cards, backdrop blur, subtle borders, layered panels, and readable contrast. |
| Claymorphism | Soft, inflated, rounded UI with playful depth and friendly colors. | Good for onboarding, mobile apps, and friendly consumer tools. | Figma community and modern UI inspiration sites. | Use large rounded cards, soft shadows, colorful gradients, and approachable spacing. |
| Neomorphism | Tactile soft-shadow UI where controls look raised or pressed into the surface. | Useful for calm settings panels and low-noise control interfaces. | https://neumorphism.io | Use restrained shadows, rounded controls, low-contrast surfaces, and avoid weak accessibility. |
| Maximalism | Bold layouts using large typography, strong color, layered visuals, and expressive composition. | Makes landing pages and launches memorable. | Awwwards, Dribbble, design inspiration libraries. | Use oversized headings, asymmetry, strong contrast, expressive sections, and bold brand rhythm. |
| Brutalism | Raw, high-contrast, direct layouts with strong type and minimal polish. | Creates personality and editorial strength. | Design galleries and editorial web references. | Use heavy typography, sharp grids, exposed structure, and intentional roughness. |
| Aurora UI | Ambient gradient light fields and soft animated backgrounds. | Gives AI interfaces a premium, atmospheric feeling. | Modern SaaS and AI landing page references. | Use gradient backgrounds, soft glow layers, restrained animation, and strong foreground readability. |
| Material Design 3 | Google-backed design system for structured, accessible interfaces. | Strong for Android, enterprise, and clear product systems. | https://m3.material.io | Use Material structure, tokens, surfaces, elevation, and interaction rules. |
| Apple Human Interface | Apple platform guidance for native-feeling interfaces. | Important for iOS, macOS, iPadOS, watchOS, and visionOS. | https://developer.apple.com/design/human-interface-guidelines | Use native spacing, navigation, controls, platform patterns, and restrained motion. |
| WCAG | Accessibility guidelines for web content and interfaces. | Prevents unreadable or unsafe UI decisions. | https://www.w3.org/WAI/standards-guidelines/wcag | Require contrast, keyboard access, focus states, labels, and readable structure. |

## Rule

Every pattern must be connected to intent, surface, and user benefit.
