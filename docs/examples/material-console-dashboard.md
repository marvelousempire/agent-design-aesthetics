# Material Console Dashboard Example

## Surface

Operator console or admin dashboard

## Intent

Support reliable, structured work for users who need clarity, status, and control.

## Pattern

Material Design 3 structure with Bento-inspired grouping where useful.

## Component Sources

- Material Design 3
- Radix UI
- shadcn/ui
- WCAG

## Agent Prompt

```text
Create an operator console dashboard using Material Design 3 structure with clear surfaces, consistent elevation, accessible controls, and strong information hierarchy. Use grouped cards for status, alerts, actions, and recent activity. Keep colors restrained and meaningful. Use visible focus states, keyboard-safe controls, readable tables, and responsive layouts for desktop and tablet.
```

## Accessibility

- Make alerts readable and non-color-only.
- Use clear labels for controls.
- Support keyboard navigation.
- Keep tables readable on smaller screens.

## Validation

- [ ] Status is visible quickly.
- [ ] Actions are clearly separated from information.
- [ ] Alerts are prioritized.
- [ ] Layout supports real operator work.
