# AI Agent Application Guide

## Purpose

This guide explains how another AI agent should use Agent Design Aesthetics.

## How To Apply A Pattern

```text
1. Read the product intent.
2. Identify the interface surface.
3. Choose the best design pattern.
4. Select supporting component libraries.
5. Create a direct coding-agent prompt.
6. Add accessibility and responsiveness requirements.
7. Validate the result.
```

## Example Prompt

```text
Create a responsive AI dashboard using a Bento Grid layout with shadcn/ui cards, Radix UI accessible primitives, strong heading hierarchy, 8pt spacing, subtle glass panels, clear empty states, and restrained Motion transitions. The interface should feel premium, readable, and easy to scan. Meet WCAG AA contrast and include visible focus states.
```

## Pattern Selection Shortcuts

| Need | Use |
| --- | --- |
| Organize many features | Bento Grid |
| Make it premium and AI-native | Glassmorphism + Aurora UI |
| Make it friendly | Claymorphism |
| Make it tactile and calm | Neomorphism |
| Make it bold and memorable | Maximalism |
| Make it native to Apple platforms | Apple Human Interface |
| Make it structured and enterprise-safe | Material Design 3 |

## Agent Safety Rule

Do not let visual effects weaken readability, accessibility, or product clarity.
